How to play?

  1. Type all the text you see on the screen, before the timer runs out.
  2. If you type correctly, the text will become green.
  3. If you type incorrectly, the text will become red. Correct the mistakes asap.
  4. There are 3 difficulty levels: Easy, Medium and Hard.
  5. Clear all the levels and lets save the multiverse.
